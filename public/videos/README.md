# Video Program Unggulan

## Cara Upload Video

Upload video untuk program unggulan ke folder ini dengan nama:

1. **indiversion.mp4** - Video untuk program Indiversion
2. **aiu-mam.mp4** - Video untuk program AIU MAM (Model ASEAN Meeting)

## Spesifikasi Video

- Format: MP4 (H.264 codec)
- Resolusi: 1920x1080 atau 1280x720
- Durasi: 1-3 menit (ideal)
- Ukuran: Maksimal 20MB per video
- Aspect Ratio: 16:9 (landscape)

## Tips Kompresi

Jika ukuran video terlalu besar, kompres menggunakan:
- Online: https://www.freeconvert.com/video-compressor
- Desktop: HandBrake (https://handbrake.fr/)

Settings untuk web:
- Video Codec: H.264
- Quality: RF 23-28
- Frame Rate: 24-30 fps
- Audio: AAC 128kbps

## Alternatif: Gunakan Link YouTube/Vimeo

Jika video terlalu besar, bisa upload ke YouTube/Vimeo dulu, lalu embed link-nya.
